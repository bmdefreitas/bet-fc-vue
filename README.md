# Bet-FC

Projeto Bet-FC com Vue.JS

## Instalação do Projeto
```
yarn install
```

### Rodando o Projeto
```
yarn serve
```

### Compilando o Projeto para Produção
```
yarn build
```

### Rodando o Lint
```
yarn lint
```

### Configurações Customizadas
Veja: [Configurações](https://cli.vuejs.org/config/).
